# User Profile & API Keys Settings Page

A page for managing user account details and configuring MCP server API keys. 

**DESIGN SYSTEM (REQUIRED):**
**Style:** Glassmorphic, dark mode, tech-premium.
**Font:** Space Grotesk.
**Colors:** Midnight Blue base, Electric Blue accents.
**Shapes:** Rounded (12px), pill-shaped buttons.
**Effects:** Subtle frosted glass transparency on all containers.

**Page Structure:**
1. Navigation sidebar on the left (integrated with Dashboard link).
2. Large "Settings" header.
3. Tabbed interface: "Profile", "API Keys", "Security".
4. API Key list with "Add New Key" button and "Revoke" actions.
5. Profile form with name, email, and avatar upload (placeholder).

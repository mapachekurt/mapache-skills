# Design System: MCP Server Status Dashboard
**Project ID:** 3705215192230869092

## 1. Visual Theme & Atmosphere
High-end, tech-focused dashboard with a **Glassmorphic** aesthetic. The atmosphere is professional, sleek, and high-fidelity, using depth and transparency to create a modern feel.

## 2. Color Palette & Roles
* **Deep Space Navy (#0a0e1c):** Background color, deep and grounding.
* **Electric Blue (#135bec):** Accent color for primary buttons, active states, and highlights.
* **Frosted Glass (rgba(255, 255, 255, 0.05)):** Card backgrounds with subtle transparency.
* **Neon Green (#10b981):** Operational status indicators.
* **Pulsing Red (#ef4444):** Critical alert status.

## 3. Typography Rules
Uses **Space Grotesk** as the primary font. Large, bold headers for titles and monospaced snippets for log activity data.

## 4. Component Stylings
* **Buttons:** Capsule-shaped (round-full) with subtle gradients or glass backgrounds.
* **Cards/Containers:** Roundness set to **ROUND_TWELVE** (12px). Semi-transparent frosted glass with subtle borders.
* **Indicators:** Pulsing neon globes for real-time status.

## 5. Layout Principles
Grid-based layout for toolkits. Sidebar for global navigation. Right sidebar for notifications.
